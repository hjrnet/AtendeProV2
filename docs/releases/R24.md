# R24 - Comercializacao SaaS, onboarding e metricas admin

Status: CONCLUIDA

## Objetivo

Preparar o AtendePro para venda SaaS com cockpit comercial, onboarding guiado, metricas admin e ambiente demo/reset por perfil.

## Tasks concluidas

- TASK-R24-001: Checkout e assinatura self-service com fluxo de plano, trial, upgrade/downgrade e preparacao para gateway real.
- TASK-R24-002: Onboarding guiado de empresa por vertical, usuarios, servicos, agenda e dados demo opcionais.
- TASK-R24-003: Metricas SaaS no Admin com MRR, churn, trials, conversao, planos ativos, vertical de tracao e alertas.
- TASK-R24-004: Ambiente demo/reset por perfil para Nutri, Beauty, gestor, investidor e suporte.

## Evidencias

- Cockpit Admin SaaS com checkout/trial/upgrade preparado.
- Onboarding guiado por etapas.
- Metricas SaaS reais via `/api/admin-saas/dashboard` e `/api/admin-saas/dashboard/vendas`.
- Planos ativos via `/api/admin-saas/planos`.
- Gateway real e reset fisico de massa demo mantidos como integracoes futuras seguras.
