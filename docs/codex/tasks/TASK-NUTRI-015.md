# TASK-NUTRI-015 — Substituições e equivalências alimentares

## Release
R18 — Nutri Pro plano alimentar avançado e produtividade clínica

## Objetivo
Permitir que o nutricionista registre substituições alimentares por plano, refeição, objetivo e restrição alimentar.

## Escopo entregue
- Endpoint para listar substituições alimentares de um plano.
- Endpoint para criar substituição com alimento de origem, substituto, grupo, objetivo, restrição, quantidade equivalente e unidade.
- Persistência em `nutri_substituicoes_alimentares`.
- Client web tipado para consumo da feature.
