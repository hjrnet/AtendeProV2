# Design System — AtendePro

## Conceito visual

AtendePro deve parecer um consultório/clínica premium transformado em software:

- saúde;
- limpeza;
- confiança;
- prosperidade;
- tecnologia;
- profundidade;
- organização;
- clareza financeira.

## Direção

- Clean, mas não vazio.
- Moderno, mas não confuso.
- Profundo, mas não pesado.
- Premium, mas acessível.
- Mobile-first.
