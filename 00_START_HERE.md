# 00_START_HERE — AtendePro Profissional

## Primeiro comando

Abra o projeto no Codex ou Antigravity e envie:

```md
status
```

O agente deve ler `AGENTS.md`, `docs/codex/SESSION_STATE.md`, `docs/ROADMAP_RELEASES.md` e identificar a próxima task.

## Para executar uma task completa

```md
auto
```

## Para execução com mais qualidade

```md
multiagente
```

## Para economizar tokens

```md
economico
```

## Para executar várias tasks

```md
autopilot 3 tasks
```

ou

```md
autopilot multiagente 3 tasks
```

## Push

Nunca peça push automático. Faça push manualmente depois de revisar os commits locais.
